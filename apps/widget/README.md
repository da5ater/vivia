# Vivia Widget

## Description

Vivia Widget is the customer-facing support interface that appears on client websites. It lets visitors start conversations, receive AI-powered help, share contact details, and use voice support when available.

## Features

- Chat interface for customer support conversations.
- AI-assisted answers connected to the Vivia backend.
- Contact capture flow for identifying and following up with visitors.
- Voice interaction support for spoken assistance.
- Loading, selection, authentication, and conversation screens.
- Standalone widget experience designed for external websites.

## Role in Vivia

The widget is the first touchpoint for website visitors. It brings Vivia support directly into the customer journey and connects each interaction back to the platform backend and dashboard.
