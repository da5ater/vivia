# Vivia Embed

## Description

Vivia Embed provides the lightweight website script used to load the Vivia support widget on external pages. It handles widget injection, configuration, and initialization for client sites.

## Features

- Embeddable script for adding Vivia to external websites.
- Widget initialization from a client page.
- Lightweight loading flow focused on fast website integration.
- Demo page for previewing the embedded widget behavior.
- Separation between the host website and the widget application.

## Role in Vivia

The embed layer connects client websites to the Vivia widget. It makes the customer support experience easy to place on a page while keeping the widget logic isolated from the host site.
