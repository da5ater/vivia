# Vivia Web

## Description

Vivia Web is the main dashboard for managing the Vivia support platform. It gives teams a central place to review customer conversations, manage integrations, adjust widget behavior, and monitor the support experience.

## Features

- Conversation dashboard for viewing and managing customer messages.
- Authentication-protected platform access.
- Integration management for connected services and support tools.
- Widget settings interface for customizing customer-facing behavior.
- Dashboard navigation for support operations and account workflows.
- Shared Vivia UI components for a consistent visual experience.

## Role in Vivia

The web app is the control center of the platform. It is where internal users configure Vivia, track activity, and manage the tools that power the customer support experience.
