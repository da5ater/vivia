# Vivia Backend

## Description

Vivia Backend contains the platform data model, API functions, AI workflows, and service integrations that power the Vivia applications. It manages conversations, contact sessions, messages, widget settings, subscriptions, secrets, and support automation.

## Features

- Real-time conversation and message handling.
- Contact session management for website visitors.
- Widget settings and configuration storage.
- AI support agent workflows for automated assistance.
- Retrieval and search tools for knowledge-aware responses.
- Plugin, secret, subscription, and user management.
- Public and private API layers for platform and widget access.

## Role in Vivia

The backend is the operational core of Vivia. It coordinates data, automation, and service logic so the dashboard, widget, and embed layer can work together as one support platform.
