# Vivia Math

## Description

Vivia Math is a shared utility package for reusable calculation and helper logic used by the Vivia workspace.

## Features

- Central place for shared numeric helpers.
- Reusable utility logic for apps and packages.
- Small, focused package for calculations that should not be duplicated.

## Role in Vivia

The math package supports shared logic across the platform. It helps keep repeated calculations consistent wherever they are needed.
