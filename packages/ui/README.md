# Vivia UI

## Description

Vivia UI is the shared interface library used across the Vivia applications. It provides reusable components and styling foundations that keep the dashboard and widget experiences visually consistent.

## Features

- Shared components for buttons, forms, dialogs, navigation, and layout.
- Accessible UI primitives built for polished product screens.
- Tailwind-based styling patterns for consistent spacing, color, and typography.
- Reusable building blocks for both dashboard and customer-facing interfaces.
- Centralized design language for faster, more consistent feature work.

## Role in Vivia

The UI package keeps the product experience coherent across different applications. It gives each part of Vivia a common visual foundation while still allowing screens to be tailored to their specific purpose.
