# Vivia ESLint Config

## Description

Vivia ESLint Config provides the shared linting standards used across the Vivia workspace. It helps keep code style, quality checks, and project conventions consistent between applications and packages.

## Features

- Shared linting rules for the monorepo.
- Consistent code quality expectations across apps and packages.
- Central configuration for workspace-wide maintainability.
- Support for cleaner collaboration across the Vivia codebase.

## Role in Vivia

This package keeps the codebase easier to maintain by giving every part of the project the same quality baseline.
