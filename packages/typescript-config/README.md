# Vivia TypeScript Config

## Description

Vivia TypeScript Config provides shared TypeScript settings for the Vivia workspace. It keeps compiler behavior and project strictness consistent across applications and packages.

## Features

- Shared TypeScript configuration presets.
- Consistent compiler options across the monorepo.
- Common typing baseline for apps, backend code, and shared packages.
- Centralized configuration for easier maintenance.

## Role in Vivia

This package supports reliable TypeScript behavior across the platform. It helps every part of Vivia follow the same typing standards.
