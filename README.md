# Vivia

## Description

Vivia is an AI-powered customer support platform built around a web dashboard, embeddable chat widget, voice assistance, and a real-time backend. It helps businesses manage conversations, configure support experiences, connect integrations, and deliver automated help directly inside their websites.

## Features

- AI support conversations for customer questions and follow-ups.
- Embeddable website widget for live chat and automated assistance.
- Voice support experience powered by an AI voice workflow.
- Dashboard for managing conversations, contacts, widget settings, and integrations.
- Real-time backend services for messages, sessions, plugins, secrets, and user data.
- Shared design system for consistent UI across the platform.

## Project Parts

- `apps/web`: Main Vivia dashboard for platform users and administrators.
- `apps/widget`: Customer-facing support widget experience.
- `apps/embed`: Lightweight embed layer used to load the widget on external websites.
- `packages/backend`: Backend logic, database schema, AI workflows, and API functions.
- `packages/ui`: Shared interface components and styling primitives.
- `packages/math`: Shared utility logic.
- `packages/eslint-config`: Shared code quality configuration.
- `packages/typescript-config`: Shared TypeScript configuration.

## Platform Focus

Vivia is designed to make customer communication faster, more organized, and easier to scale. The platform combines automation with human-friendly conversation tools so businesses can support visitors across chat, voice, and dashboard workflows.
